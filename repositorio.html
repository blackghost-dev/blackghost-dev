# repoblackghost REPOSITORY

Versão Atual: 2.0

Instruções para Adicionar este Repositório ao Kodi

  1. Acesse o Gerenciador de Arquivos no Kodi.
  2. Clique em "Adicionar Fonte".
  3. Insira o seguinte endereço: https://repoblackghost.github.io e nomeie como "repoblackghost Repo".
  4. Retorne ao menu principal e vá até Add-ons.
  5. Clique em "Instalar de um arquivo .zip", selecione a fonte repoblackghost Repo e instale o arquivo: repository.repoblackghost-2.0.zip.
  6. Após a instalação, volte a "Add-ons", mas desta vez clique em "Instalar do Repositório".
  7. Selecione o "repoblackghost Repository" e explore os conteúdos disponíveis.

Agora é só aproveitar o conteúdo disponível!
